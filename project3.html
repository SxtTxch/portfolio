School Archive System

🚧 Abandoned Project 🚧

I developed a local school archive system using Firebase 🔥 and Next.js 🌐. This system allows users to manage and share educational documents with ease. My work focused on implementing user authentication, email verification, and a document publishing system.

Key features and contributions include:

  • User Authentication 🔐: Implemented secure login and registration processes with email verification to ensure user authenticity.
  • Document Publishing 📂: Enabled users to request and publish documents or archives containing images and text, such as lesson notes.
  • Search and Filter 🔍: Allowed users to select a teacher and view all published lessons associated with that teacher.
  • Database Integration 💾: Utilized Firebase for real-time data storage and management, ensuring quick access to and retrieval of documents.

This system was designed to facilitate easy access to educational resources while maintaining secure and efficient data handling. It aims to enhance the learning experience by providing a centralized archive of valuable lesson materials.
