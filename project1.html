Laser Leagues

Laser Leagues 🎮 is a close-sourced Roblox game developed with a focus on advanced gameplay mechanics and immersive user experience. In this project, I was responsible for both the front-end and back-end coding, ensuring a seamless and engaging experience for players.

My contributions included:

  • UI Handling and Animations 🎨: Created dynamic and responsive user interfaces to enhance gameplay.
  • Advanced Party System 👥: Implemented a sophisticated system for party management and social interactions.
  • Missions System 🎯: Developed a comprehensive missions system to keep players engaged with challenging tasks.
  • Mailbox 📬: Integrated a mailbox system for in-game communication and notifications.
  • Capsule Opening System 🎁: Designed an interactive capsule opening feature for rewards and collectibles.
